# OneIndex-moe-theme
一个对 OneIndex 自带 nexmoe 主题的修改

## 使用方法
1. 在 OneIndex 的目录下的 view 文件夹中新建一个文件夹
2. 将文件全部拷贝进去
3. 如果需要顶栏，就删除 layout-no-header.php，不需要则删除 layout.php,并将 layout-no-header.php 改为 layout.php
4. 在后台主题选项中选择新建的文件夹名称并保存
